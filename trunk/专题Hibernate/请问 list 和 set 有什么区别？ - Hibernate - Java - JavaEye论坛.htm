在Hibernate里面的List和Set同Java Collection里面的List和Set的概念是一样的。
Set声明的one-to-many中，many那一方没有重复的元素。List声明的one-to-many中，many那一方可以重复，并且有索引，因此需要在many那一方的表中增加一个字段用来保存索引。
