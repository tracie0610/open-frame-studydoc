The documentation is quite good concerning the request built with criteria, but I cannot succeed to perform a request with count.
I have a poor database for testing with 3 tables :
Book, Author, and Writtenby because a Book can be written by several authors.
I generate the java files : Book and Author with respectively a Set of Authors and Books.
Now, I did not find a way to execute this simple native SQL request :
select book.title, count(writtenby.author_id)
from book, writtenby
where book.id=writtenby.book_id
group by 1
For the moment, I wrote this piece of code

Criteria crit = session.createCriteria(Book.class);
crit.setFetchMode("Author", FetchMode.JOIN);
ProjectionList projList = Projections.projectionList();
projList.add(Projections.groupProperty("title"));
projList.add(Projections.count("authors"));
crit.setProjection(projList);
List list = crit.list();

Look there:

I solved the problem using Projections.
My code is like that:
Criteria criteria = session.createCriteria(getReferenceClass()); criteria.setProjection(Projections.rowCount());
return ((Integer)criteria.list().get(0)).intValue();
