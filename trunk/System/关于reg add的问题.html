    首先一点你要明白注册表里面%XXXX%只是在REG_EXPAND_SZ下存在的

    但是path的原始数值是REG_SZ

    如果你要强行写进去，那就按照下面的

    批处理里面的写法：

    reg add "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment" /v "path" /d "%path%;%%"SYSTEMROOT"%%" /f

    直接敲命令的写法：

    reg add "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment" /v "path" /d "%path%;%"SYSTEMROOT"%" /f

    BatchFile code
    @echo off
set MY_PATH=C:\test
for /f "skip=4 tokens=1 delims=" %%a in ('reg query "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment" /v path') do set ipath=%%a
set ipath=%ipath:path=%
set ipath=%ipath:REG_SZ=%
set ipath=%ipath:  =%
set ipath=%ipath:        =%
reg add "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment" /v path /d "%ipath%;%%MY_PATH%%" /f
