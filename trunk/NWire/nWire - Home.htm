Java Code Exploration Evolved

nWire is an Eclipse™ plug-in which expedites Java development. Its' main purpose is to help developers find code quicker and easily understand how it relates to the rest of the application, thus, understand the application structure. nWire simplifies and accelerates the coding of compound applications, reducing development and maintenance costs, by presenting one unified picture of the component and association in the application. It assists in improving the quality of the code and maintaining correct architecture. nWire emerged from over 20 years of software development and design experience. It represents a vision: to assist developers in maximizing their abilities with simple and practical tools.

nWire offers unique features

  • Browse components and associations using one simple unified view
  • Incremental search for any type of component, including classes and methods
  • Interactive visual representation of all code associations

Answer for today's challenges

Modern applications, based on object-oriented languages, adhere to standard architecture and design patterns. The application is built using small and simple building blocks, wired together in a complex design. The challenge is understanding how the single component fits within the architecture and how it relates to the rest of the system.

Failure to understand the architecture will result in the bugs. Such bugs may not immediately affect the functionality of the system, however, they will degrade the performance, lower scalability, increase system vulnerability and increase maintenance costs. Discovering these bugs later results in higher price for fixing them.

nWire is different

An industry first, nWire analyzes all the application components and associations, storing them in a single repository which is easily browsed and queried. With initial support for Java components, future releases will add support for components defined in popular application frameworks like Spring, as well as support for additional programming languages.
