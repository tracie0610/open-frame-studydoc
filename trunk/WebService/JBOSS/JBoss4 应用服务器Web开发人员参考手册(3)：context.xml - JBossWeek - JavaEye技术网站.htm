JBoss4 应用服务器Web开发人员参考手册(3)：context.xml

context.xml文件定义了所有Web应用的缺省Context元素. 它的属性如下:

  • cookies: 该标志表示sessions是否使用cookies跟踪实现. 缺省值为 true.

  • crossContext: 该标志表示ServletContext.getContext(String path)方法是否能够返回部署在相同虚拟主机中其它Web应用的contexts.

你可以在Tomcat文档中找到Contex的更多信息： http://jakarta.apache.org/tomcat/tomcat-5.5-doc/config/context.html.
