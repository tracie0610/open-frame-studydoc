    恢复GDM

    12 05月 2008

    有时由于更新不完整的原因，或是显示刷新频率的问题导致GDM启动不了。可以如下恢复。
    安装光盘源，恢复启动GDM
    sudo apt-cdrom add
    sudo apt-get install gdm
