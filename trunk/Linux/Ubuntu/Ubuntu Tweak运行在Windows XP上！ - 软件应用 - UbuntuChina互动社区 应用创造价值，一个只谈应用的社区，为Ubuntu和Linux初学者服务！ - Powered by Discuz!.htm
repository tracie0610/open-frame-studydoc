Ubuntu Tweak运行在Windows XP上！

继Ubuntu Tweak运行在Fedora、Gentoo下以后，Ubuntu Tweak如今又正式登陆Windows XP啦！
多亏了Portable Ubuntu这个伟大的项目。
看图后再说话：


Windows XP的窗口边框＋Ubuntu的主题控件样式。

Portable Ubuntu实际上是基于Colinux之上的，号称不需要装虚拟机，就可以在Windows下同时运行一个Linux。
事实上也是通过一种特殊的虚拟机形式，通过后台启动它，并用X Server的形式将Linux的窗口显示在标准的Windows窗口中。这跟VMware、VBox的原生模式类似。
上图的系统系统也显示了这些，Ubuntu系统被分配了250MB的内存（我的机器是2GB）。硬盘也是独立的与Windows分区不相互干扰的（不过C盘被挂载到了/mnt/C下）。
总的来说，如果想快速地体验下Ubuntu，而不想装虚拟机之类的，玩玩Portable Ubuntu也不错！
