Ubuntu系统更换登录界面实例图文教程

http://blog.sina.com.cn/s/blog_574fa1f00100db4v.html

在Ubuntu系统里，它的“登录画面”，英文叫做：GDM themes，在下面的图中就能看到。   首先从：http://www.gnome-look.org/index. ...a0a8f5985528d985c86，下载登录画面的，主题包。这次的例子，我用的是里面的：Simple Ubuntu 主题。如图
   

    点击其中的“Simple Ubuntu 8.102.1“，进入一个下载页面，点击其中的”Download“，就能自动开始下载的。（请别告诉我你不认识 Download，这个单词…………）
   

   点击后，如果这个主题能正常下载，等待10秒左右，浏览器就能自动开始下载了。如果等半天还没开始下载，就去点击网页中间的： Clickhere，就能开始下载的。
  

   我是下载到了桌面，下载后是这样的：
   

留着先，不管它了。（千万别双击打开，不是这样安装的！ubuntu下的任何主题包，都别双击打开！）

   从你的任务栏，依次：System（系统）——Administration（系统管理）——Login Window（登录窗口）。
    点击上面的“Local （本地）“，就来到了：登录窗口主题设置界面。如图
  

    看到图了吧？点击里面的 “Add（添加）“，就能添加主题了，点击后，弹出文件选择对话框，选择刚才下载好的主题包，点击”install（安装）“

   安装后，自动返回主题设置界面，如图，在“Add“按钮的左边，就能找到你安装好的主题了。用鼠标选择它即可更换！
  

   行了，关闭界面吧，更换完成。你现在可以按下： ctrl +alt + Backspace（回格），组合键，注销试试看了。一般都没问题了。
   可能出现的问题：
   如果你注销后，UB的登录界面并没有改变，就回到登录界面设置窗口，把“Theme（主题）“这里，后面的项目，改成： Selectedonly 只选择的“，就行了。 UB有这个小毛病。
