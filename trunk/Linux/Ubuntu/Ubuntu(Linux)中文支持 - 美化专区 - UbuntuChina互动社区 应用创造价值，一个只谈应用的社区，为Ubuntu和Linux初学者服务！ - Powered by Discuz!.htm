Ubuntu(Linux)中文支持

一、桌面的中文支持
其实，在系统中已经做好的，我们要做的只是从网上下载安装好就行。对于不习惯使用命令的朋友可以直接点击系统(system)->系统管理(administration)->语文支持(languagesupport)，然后在上面“支持语言”的选框中选中“Chinese”,并在下面“默认语言”下拉列表中选中“Chinese(China))，点击确定即可，系统会自动下载并安装好文件。

对于，在系统安装时语言选择中国（Chinese)的朋友，系统默认安装了SCIM中文输入法，但是没有输入码表，是不能输入汉字的，我初用Ubuntu的，所以当初我也不知道这个问题，就傻傻地切换着输入法。经过语言选择操作就会自动安装好拼音的码表

[ sudo apt-get install language-pack-zh language-pack-zh-baselanguage-support-zh ]
[ sudo apt-get install scim scim-gtk2-immodule scim-modules-socketscim-modules-table scim-pinyin scim-tables-zh scim-input-pad]（其中带table就是码表表了）
当然对于scim输入法有些小小的bug比如不能改文件名，虽然有解决之道，但是我始终觉得不舒服，所以我选择了fcitx（以前输错这个名字，所以在安装时经常找不到包，安装指令如下：
[
sudo apt-get install im-switch fcitx
sudo im-switch -s fcitx -z default
im-switch -s fcitx -z default #注意：不加sudo
]

完成设置最好重启一下X，输入法就生效了。某些情况下可能，在安装了fcitx输入法以后可能会出现和SCIM并存的问题，解决方法如下：
[sudo gedit/usr/lib/gtk-2.0/2.10.0/immodule-files.d/scim-gtk2-immodule.immodules]
将内容改为如下：
# automatically generated by dh_gtkmodules, do not edit
“/usr/lib/gtk-2.0/2.10.0/immodules/im-scim.so”
“scim” “SCIM Input Method” “scim” “/usr/share/locale” “ja:ko”
#就是将最后一行中的"ja:ko:zh"修改为"ja:ko",这样修改以后，scim在中文环境下将不被启动

二、中文ManPagne
很多人如我，所以希望执行指令的帮助文件是中文环境就好了，呵呵，在网络上有这么个工程的。并且有已经翻译好，并加入源里，需要执行如下指令：
[sudo apt-get install manpages-zh]
三、文本模式中文化
默认情况下，Ubuntu启动已经紧密集成X，以前学习Linux时候接触过inintab配置文件(好像有这么个东东），但在Ubuntu下已经找不到这个了，但提供了一个小工具来修改启动配置，以决定是否在启动时进入文本模式。安装这个工具需要执行如下命令
[ sudo apt-get install sysv-rc-conf ]
[sudo sysv-rc-conf]
找到gdm项，然后在运行级别2列按空格取消X标志，（我猜测Ubuntu默认运行级别为2），按q退出，以后进入系统就是文本模式的了哦，要想启动X可以输入指令
[startx]
当然你也可以按CTRL+ALT+F1-F6来进入文本模式哦，文本模式的，文本提示符的显示控制由~/.bashrc来控制，你可以自由定制（这可是我找了好久才找出来的）。文本模式下比较好的中文支持是zhcon，你安装它必须输入指令：
[sudo apt-get install zhcon]

在运行zhcon时必须注意加载vga驱动和utf8支持，如果没加VGA会黑屏的，到时只有Reset，所以你的命令行应该是：
[zhcon --utf8 --drv=vga]
对于zhcon本来是可以切换输入法的，但是与utf8有冲突，导致了某些按键无效（采用网上某个朋友的说法，我比较相信），所以只能用英语和其他一种输入法（Ctrl+space没有冲突）。通过阅读/etc/zhcon.conf文件可以知道，zhcon的配置文件可以有两份，一份是/etc/zhcon.conf（公用），另一份则是~/.zhconrc（私用，优先权高于公用那个，如果没有可以拷贝一个/etc/zhcon.conf副本
[~$ cp /etc/zhcon.conf ~/.zhconrc]
就可以了）。并且zhcon可以有9种输入法，在输入法列表中的前9种输入法可用。但事实并非如此，只有排第一位置的输入法可用（冲突问题，输入法切换热键冲突，真希望开发者能够改改），当然聪明的你知道把你要用的输入法行拷贝到列表的第一位置了，我采用加#号屏蔽的，这样不会出错。呵呵，应该有几个字符认不出来，万一拷贝掉了一个，配置文件乱码就麻烦了。每次都输入[zhcon --utf8--drv=vga]也太麻烦了，并且一不小心忘打了就黑屏了哦，所以在~/.bashrc里面加一个别名就方便多了。通过
[~$alias]
可以知道Ubuntu的alias命令格式为
[alias别名='命令+参数']
所以在~/.bashrc末添加一条
[alias zhcon='zhcon --utf8 --drv=vga']
以后再次输入zhcon不会黑屏了。

通过中文支持安装后，很多地方都变成中文的了，比较适合我用了。看看Ubuntu是不是做得很好呢？请记住免费使用许可哦。今天接到一个任务，将纸质的图书名册电子造册，我打算用OOo的数据库来完成，但是我没有OOo数据的知识，不过试试“想想就能成，试试就能赢”嘛，加油哦！
