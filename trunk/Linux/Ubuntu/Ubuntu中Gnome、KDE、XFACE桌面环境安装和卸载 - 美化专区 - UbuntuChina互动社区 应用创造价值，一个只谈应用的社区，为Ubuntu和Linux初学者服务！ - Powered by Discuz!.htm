Ubuntu中Gnome、KDE、XFACE桌面环境安装和卸载

一）在终端中运行安装：
 1、安装XFACE： sudo apt-get install xubuntu-desktop 点上面的菜单“应用程序－附件－终端”进入到终端，先输入你的用户的密码
2、安装KDE：sudo apt-get install kubuntu-desktop
 使用方法同上
3、安装Gnome：sudo apt-get install ubuntu-desktop
 使用方法同上
（二）在新立得中安装： 1、点上面的菜单“系统－系统管理－新立得软件包管理器”，输入你的用户密码，
 2、在窗口右边点“搜索”，分别以xubuntu-desktop、kubuntu-desktop和ubuntu-desktop搜索， 3、在搜出的列表中，点右键选“标记安装”， 4、点工具栏的“应用”按钮，再点“确定”即可安装 安装好以後，在登录对话框中，选择相应的桌面环境
卸载桌面环境（应备份重要文件）

1、卸载gnome：sudo apt-get --purge remove liborbit2 2、卸载kde：sudo apt-get --purge remove kdelibs4c2a libarts1c2a
 3、卸载xface：sudo apt-get --purge remove xfce4




1、安装KDE桌面
KDE是一个很流行的，功能全面的桌面环境。KUbuntu就是一个Ubuntu的一个使用KDE桌面的版本。
安装和使用KDE：
1）使用Ubuntu新立得软件包管理器安装K桌面，或者使用APT安装KDE（sudo apt-get install kde）
2）注销系统，在登录屏幕选择'选项'－'登录会话'，选择KDE，登录即可。

2、安装Xface桌面
Xface是一个快速轻量级的桌面，XUbuntu 就是一个Ubuntu的一个使用Xface桌面的版本。
步骤与KDE安装雷同。
