让开源“电驴”在Ubuntu系统上飞奔

http://os.yesky.com/349/8676349.shtml

许多朋友有在Windows平台上用“电驴”软件(如Emule)下载喜欢的电影、音乐、软件的经历。在Ubuntu系统中，使用开源的“电驴”软件Amule，一样可以让我们轻松下载网上资源。在Ubuntu 8.10桌面操作系统中，安装、配置和使用Amule只需几步操作，我们不妨一起来看一看。
　　第一步：安装Amule
　　Ubuntu系统默认没有安装Amule软件，我们可以使用apt-get命令或新立得软件包管理器来安装它。如使用apt-get命令，则在“应用程序”→“附件”下找到“终端”项，单击该项，则打开一个GNOME终端窗口，在该窗口中输入如下命令：
　　sudo apt-get install amule amule-utils
　　命令执行后，将自动在线下载并安装Amule软件，当前下载的Amule软件的版本为2.2.2。另外，我们也可以从Amule的官方站点www.amule.org下载最新的软件包来安装。软件安装完毕后，在菜单“应用程序”→“互联网”下可以找到“Amule”项，点击该项，则运行Amule程序。
　　下载：http://www.mydown.com/soft/network/download/239/439739.shtml
　　第二步：简单配置Amule
　　安装好Amule软件后，第一次运行Amule程序时，程序提示是否需要下载服务器列表，我们点击“是”按钮，则Amule自动下载服务器列表文件。进入Amule主窗口后，点击窗口上方的“连接”按钮，我们就可以用Amule下载网上资源了。不过，为了让Amule能顺畅下载网上资源，我们可以手动添加一些服务器列表文件或Kad节点文件。另外，还要手动建立Firefox浏览器与Amule程序的关联。
　　1.下载ed2k服务器列表文件
　　点击Amule窗口上方的“网络”按钮，在“ED2K”选项卡中，手动添加如下链接：
　　http://www.emule.org.cn/server.met
　　2.添加Kad的节点文件
　　在“Kad”选项卡中，手动添加如下链接：
　　http://renololo1.free.fr/e/nodes.dat
　　3.关联Firefox与Amule
　　由于Amule程序没有自动与Firefox关联，因而如果想要在Firefox中点击下载资源的链接，自动将下载任务添加到Amule的下载任务列表中，还得手动对Firefox进行配置。
　　运行Firefox浏览器，在地址栏中输入命令：about:config，命令输入后回车，则在地址栏下方的窗口中列出Firefox的配置选项。点击鼠标右键，在弹出的快捷菜单中选择“新建”→“布尔”项，则新建一个布尔值(Boolean)项，在弹出的对话框的“输入首选项名称”栏内填写“network.protocol-handler.external.ed2k”，在随后弹出的对话框中选择值“true”。再新建一个布尔值(Boolean)项，首选项名称为“network.protocol-handler.expose.ed2k”，值为“false”。新建一个字符串(String)项，首选项名称为“network.protocol-handler.app.ed2k”，值设为“/usr/bin/ed2k”。设定完毕后，重启Firefox浏览器。
　　第三步：用Amule下载资源
　　用Amule下载电影、音乐、软件等网上资源非常方便。常用的下载资源的方式有三种：一是利用Amule的搜索器查找网上资源并下载;其二是使用Firefox浏览器查找到资源后，复制下载链接到Amule中进行下载;其三是在Firefox浏览器中直接点击下载资源的链接来进行下载。这里以第一种和第三种方式为例来介绍。
　　打开Firefox浏览器，输入网址：http://www.verycd.com，在该站点搜索到所需的资源，如搜索到电影“桃花运”，进入该电影的详细页面后，将会看到电影文件下载框(图1)，　直接点击“下载选中的文件”按钮，则将下载资源的所有文件均添加到Amule下载任务列表中。另外，我们也可以点击某一个文件的链接来添加单个文件的下载任务。


　　Amule 软件内置搜索功能，点击Amule程序窗口上方的“搜索”按钮，则出现资源搜索窗口。在“名字”输入框中输入下载资源的关键字，如搜索港版电视剧《上海滩》，则输入关键字“上海滩”，点击“开始”按钮进行搜索。搜索到相关资源后，选中资源列表中的某一项，并点击“下载”按钮，则Amule自动添加该资源的下载任务，点击窗口上方的“传输文件”按钮，可看到当前所有下载任务的详细情况(图2)。下载的资源默认保存到/home/username/.aMule/Incoming目录下，其中username为当前登录用户名。
