=================美化套件指北=======================
系统自带的主题（套件）推荐：

Human，薄雾，滑翔机和十字架

新立得安装GNOME官方美化组件（比较省事的方法）
或终端下
代码:
sudo apt-get install gnome-art

或点击 [url=apt://gnome-art/]apt://gnome-art[/url]
运行 gnome-art

官方软件仓库里的主题（套件）：
适合日常使用的Peace和Tropic和Blubuntu主题以及酷炫的深蓝色Ubuntu Studio主题套件，穆斯林的主题套件 ---绿色的Ubuntume
到 ”新立得软件包管理器“ 里搜索名字就能找到， 每个都有5项

天蓝色的Blubuntu主题套件
代码:
sudo apt-get install blubuntu-wallpapers blubuntu-theme
blubuntu-session-splashes blubuntu-gdm-theme blubuntu-look

暗红色的 Peace主题套件
代码:
sudo apt-get install peace-wallpapers peace-theme
peace-session-splashes peace-gdm-theme peace-look

橘红色的 Tropic主题套件
代码:
sudo apt-get install tropic-wallpapers tropic-theme
tropic-session-splashes tropic-gdm-theme tropic-look

Ubuntu Studio

酷炫的Ubuntu Studio主题套件
代码:
sudo apt-get install ubuntustudio-theme
ubuntustudio-icon-theme ubuntustudio-gdm-theme
ubuntustudio-wallpapers usplash-theme-ubuntustudio


另外特别推荐穆斯林的主题套件 ---绿色的Ubuntume
[url=apt://ubuntume-themes/]apt://ubuntume-themes[/url]
代码:
sudo apt-get install ubuntume-themes

合理改动一下，这也是个非常好的主题－－－主要是鼠标指针旁边的字比较扎眼

适合日常使用的GTK主题（非套件）

Silicon
[url=apt://silicon-theme/]apt://silicon-theme[/url]
代码:
sudo apt-get install silicon-theme


Legacy Human
[url=apt://legacyhuman-theme/]apt://legacyhuman-theme[/url]
代码:
sudo apt-get install legacyhuman-theme


New Wave、Kin、Dust
包含在Community themes里
[url=apt://community-themes/]apt://community-themes[/url]
代码:
sudo apt-get install community-themes


外部主题：

Mac4Lin的GNOME套件（包含如何使用的 PDF
文档）该主题套件主要是为了模仿出Mac效果
http://sourceforge.net/project/platform ... _id=204373

deb  http://ubuntusatanic.org/hell  hardy  
main

钥匙：代码:
wget  -q  http://ubuntusatanic.org/ubuntu-se-key.gpg

===================美化桌面工具指北=======================


精致时钟

代码:
sudo apt-get install cairo-clock


Dock

代码:
sudo apt-get install avant-window-navigator
awn-manager


桌面监视conky

代码:
sudo apt-get install conky


Conky 默认并不会创建配置文件，需要使用下列命令手动生成：

代码:
zcat /usr/share/doc/conky/examples/conky.conf.gz >
~/.conkyrc


相关参考：Conky
Variables

Linux版Google Gadgets

8.04可以加入以下源：
代码:
deb http://ppa.launchpad.net/googlegadgets/ubuntu hardy main
deb-src http://ppa.launchpad.net/googlegadgets/ubuntu hardy
main

然后
代码:
sudo aptitude update & sudo aptitude install
google-gadgets

安装完后运行:
代码:
ggl-gtk -s -bg
