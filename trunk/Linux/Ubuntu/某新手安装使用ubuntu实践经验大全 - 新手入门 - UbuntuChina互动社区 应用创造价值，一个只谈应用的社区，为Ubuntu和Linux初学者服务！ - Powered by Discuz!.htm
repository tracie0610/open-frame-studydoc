某新手安装使用ubuntu实践经验大全

http://blog.sina.com.cn/s/blog_4 ... type=v5_one&amp;label=rela_prevarticle

最近开始接触Linux，虽然使用起来没有Windows方便，但感觉还是挺不错的，至少不用担心会中毒了，如果中毒的话那一定要去买彩票～～

因为学习需要，把Windows卸载了，开始的时候在一个大牛的帮助下安装了Gentoo，那是相当痛苦的，仅安装就用了两天时间。两天后终于出现图像界面了，不过竟然无法登录，实在是痛苦。几经周折终于登进去了，不过没声音，而且显示效果也不是太好，本来还想修复一下，不过第二次开机的时候就不能上网了，无语～～～最后选择了ubuntu，用起来好多了

ubuntu的安装比较简单，有友好的安装界面，不像Gentoo那么变态。唯一可能要注意的就是分区，如果选择手动分区，那么交换分区（swap）可以设置成内存的1－1.5倍，虽然大部分利用不起来，不过现在硬盘都大的很，也就不必要计较那点儿空间了。／根分区和／home分区差不多四六分就好了。

在安装系统的时候最好拔掉网线，因为在线更新是很浪费时间的。安装好之后，要更新一下源，在终端输入“sudo nano/etc/apt/sources.list”，把sources.list的内容替换成一个比较好的源的地址，保存关闭就好。然后打开系统管理中的更新管理器，把包更新一下就好了，它会自动解决依赖关系。如果源好的话，速度会非常快。更新之后设置一下语言支持，就可以让系统支持中文了。Linux的驱动是嵌入到内核中的，不过显卡的驱动还要装一下，这样会使显示效果更好一些。

使用ubuntu已经一周了，用起来已经有点儿顺手了。ubuntu下的软件资源还是比较丰富的，输入法可以选择SCIM（ubuntu下安装软件比较简单，可以先试一下“sudoaptitude install 软件名”如：sudo aptitude installscim，如果源内没有资源的话再从网上下载deb包）。用惯了QQ和飞信，ubuntu下当然也是不能少的，QQ可以用QQ forLinux，现在已经支持备注显示了，基本功能也都有。飞信可以选择Libfetion。下载工具如果用迅雷的话，那就要用国际版的了，不过之前要安装一下Wine，用起来虽然没有迅雷5好用，但还是不错的。如果想用XP虚拟机，那可以安装一个virtualBox，安装增强功能要先从网上下载VBoxGuestAdditions_2.0.4.iso，然后重命名为VBoxGuestAdditions.iso放到文件夹/usr/lib/virtualbox/或/usr/spare/virtualbox/下，这过程中要对文件夹virtualbox进行访问授权（授予root权限sudochmod 777 /usr/lib/virtualbox/），在安装之前一定要记得收回权限（收回root权限sudo chmod555 /usr/lib/virtualbox/），不然虚拟机将无法启动。至于娱乐的话就安装一个终极播放软件GNOMEMplayer吧，用起来非常爽

最近在使用ubuntu，并且用 VirtualBox装了一个XP虚拟机，这样用起来方便一些。但需要解决ubuntu与XP的通信问题。这时可以有多种选择，例如使用U盘（这要修改 UBS的使用权限），建FTP（这个麻烦一点儿），其实最简单的还是利用文件共享，几分钟就能搞定。首先安装增强功能（VBoxGuestAdditions），然后设置VirtualBox配置中的数据空间，选定一个ubuntu要共享的目录,然后进入虚拟机重装VBoxGuestAdditions，重启之后。右键点击我的电脑，打开映射网络驱动器，点击浏览就可以找到ubuntu下的共享文件夹了。打开我的电脑，会在下方出现“网络驱动器”，这时就可以用了。另外“登录时重新连接”要选择一下，这样每次登录虚拟机就可以自动连接了。

如果想用Ubuntu，而没有LiveCD，那可以选择使用U盘安装。安装方法特别简单，而且速度超快，下面介绍一下安装方法：

1、下载最新版本的Grub4DOS
下载地址： http://download.gna.org/grub4dos/grub4dos-0.4.4-2008-08-08.zip
下载并解压缩后，将目录中的grldr （非grldr.mbr），grub.exe两个文件复制到C盘根目录下

2、下载好的iso文件中解压出casper文件夹，找到vmlinuz、initrd.gz并复制到C盘根目录下

3、C盘根目录下建立menu.lst文件，内容为：
title Install Ubuntu
root (hd0,0)
kernel (hd0,0)/vmlinuz boot=casper iso-scan/filename=/ubuntu-8.10-desktop-i386.iso ro quiet splash locale=zh_CN.UTF-8
initrd (hd0,0)/initrd.gz

4、接着，在我的电脑–>工具–>文件夹选项–> 的查看标签下去掉“隐藏受保护的操作系统文件”之前的勾，并勾选“显示所有文件和文件夹”。取消C盘根目录下的boot.ini文件的“只读”属性，然后用记事本打开boot.ini文件，做如下更改：timeout=0 改成 timeout=5 或者更大的数字，在boot.ini 文件内容末尾加上一行 C:\grldr="GRUB"
（附：boot.ini 文件路径 c:\boot.ini ）

5、将ubuntu-8.10-desktop-i386.iso复制或移到U盘根目录下，硬盘上原有的ISO文件则修改文件名。

6、插上U盘重启电脑，选择进入Windows系统，进入后会提示进入Windows系统还是进入GRUB，选择GRUB就可以开始安装了

在安装Debian系统时，强烈建议使用网络镜像，教育网内的可以使用debian.ustc.edu.cn(列表中没有，选择手动输入，把debian.ustc.edu.cn输进去就可以了)，刚试过的，速度不错。


如果在安装debian基本系统的时候选择了英文的语言环境，之后又希望将它改为中文，需要进行以下工作

1、设置locale
2、如果没有安装locale工具，则首先要安装它，然后执行 #dpkg-reconfigure locales
3、选择语言包。需要添加上zh_CN开头的几个选项，当然你也可以只添加一个，然后，选择其中的一个作为默认。设置完成后，执行locale命令检查一下当前的locale设置是否为中文了

4、安装中文字体
中文环境当然需要中文字体的支持，手工添加中文的xft字库，另外一篇文章里已经讲过，另外我们也可以使用apt安装几个免费的中文字库。如：
#apt-get install ttf-arphic-gbsn00lp
#apt-get install ttf-arphic-gkai00mp


重启X，如果我们重启X后，发现整个系统的界面显示的还是英文，而使用locale命令检查系统的locale设置又已经正确的配置为中文。我当时遇到的情况就是某些应用的标题等显示为中文，但是工具菜单等显示和文件管理器等都显示成了英文。这种情况很有可能是因为/etc/enviroment文件里有关语言的环境变量的设置不对。打开/etc /enviroment文件，通常会发现下面两个字段：
LANGUAGE=“en_US:en”
LANG=“zh_CN”
在LANGUAGE字段添加中文的设置，比如：
LANGUAGE=“zh_CN:zh:en_US:en”
重启X，系统的语言环境应该就变为中文的了。

如果使用网络镜像的话，语言包可以自己给下载好，就省去一些麻烦。
