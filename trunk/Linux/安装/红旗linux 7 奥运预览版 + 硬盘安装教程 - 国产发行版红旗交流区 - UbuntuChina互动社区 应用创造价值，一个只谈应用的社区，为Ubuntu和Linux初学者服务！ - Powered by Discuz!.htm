红旗linux 7 奥运预览版 + 硬盘安装教程

http://hi.baidu.com/smartkeen/bl ... ccc253b219a8a7.html

如今发布的红旗linux 奥运预览版已经可以从Fat32 格式安装啦。
1、红旗linux 奥运预览版(单CD)下载及介绍：
官方下载页面：http://www.redflag-linux.com/xiazai/xiazai.php?id=1437
红旗Linux奥运预览版是红旗Linux向北京奥运会的献礼产品。
当然，该版本作为红旗Linux桌面发行版测试阶段的产品，还有很多不成熟的地方有待完善，希望获得您的谅解。
对于您对红旗Linux及其产品的关注，我们表示诚挚的感谢！也期待您在感受红旗Linux奥运预览版从安装到使用界面所带来的奥运风的同时，能对红旗Linux下个桌面发行版产品 (Version 7.0)有一初步了解。若您还有兴趣将意见和建议反馈回来，我们将珍视这笔可贵的财富！
系统亮点：
LiveCD（本版本只支持LiveCD方式的预览和安装，其他安装方式暂不提供） KDE 4.1 奥运主题Windows集成（系统启动后，可通过欢迎向导中该工具将Windows用户相关信息及配置导入进来；它将Windows的Administrator视作root账户，并可导入账户相关属性，其中包括密码、我的文档、墙纸、分辨率设置和网络配置等信息。值得注意的是，作为Beta2阶段，本版本还不支持Windows Vista系统。）
组件信息：
kernel 当前为2.6.25 KDE 4.1 firefox 3.0 glibc 2.8 gcc 4.3.1 qt 4.4.0 xorg-x11-server 1.4.99 wine 1.0
问题澄清： 声卡驱动问题比较多，可能很多情况下没有声音 Intel集成显卡在通过图形方式设置分辨率时，有出现崩溃的可能 集成的媒体播放软件运行不够稳定
MD5: ab31d983cc16ff760c165db6666125e2 olympic7.iso
请选择最近的链接下载：

http://60.28.182.153/data/dt7/olympic7.iso

2、把下载的红旗linux 奥运预览版iso 解压到fat32 分区的根目录下。本教程是把它放在C 盘目录下的，如你不是如此,请修改相应menu.lst 中的内容。
3、设置grub4dos 引导列表。先下载grldr，然后书写menu.lst 引导列表。再在windows xp/2000 的boot.ini 添加grub4dos 引导选项。
（1）下载grldr文件，把下载得到文件中grldr，menu.lst解压到C 盘根目录下。使用其中boot.ini 作为参考便可。不要随意覆盖原文件，最好是在原文件的基础上编辑，下文有介绍。
grldr 下载地址：http://blogimg.chinaunix.net/blog/upfile2/080810170159.rar
（2）打开c:\boot.ini，编辑boot.ini，在boot.ini最后添加一行：c:\grldr="红旗linux 奥运预览版安装"，以下是我的boot.ini文件内容，可做参考。
[boot loader]
timeout=15
default=multi(0)disk(0)rdisk(0)partition(1)\WINDOWS
[operating systems]
multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /noexecute=optin /fastdetect
c:\Arldr="DOS工具箱"
c:\wubildr.mbr="Ubuntu"
c:\grldr="红旗linux 奥运预览版安装"
（3）核对编辑menu.lst，红旗linux 预览版iso 解压到C 盘的menu.lst 内容如下：
       default 0
       timeout 4
       color yellow/brown light-green/black       title redflag Chinese(Simplified)
       root (hd0,0)
       kernel /isolinux/vmlinuz0 root=/dev/sda1 rootfstype=vfat ro quiet liveimg vga=788 live_locale=zh_CN.UTF-8
       initrd /isolinux/initrd0.img        title BacK To windows Loader
       rootnoverify (hd0,0)
       makeactive
       chainloader +1
如果你把iso 文件解压到其他fat32 盘里，就得修改上面红色部分了，例如你的fat32 是D 盘，就修改(hd0,0)为(hd0,4)，sda1 改为 sda5；若是E，F 盘，以D 盘依次加1 便是。
最后，一切OK 了，就可以使用红旗linux 奥运预览版的livecd 了，关于如何安装，进入livecd 后，一切都是那么简单，按照中文向导做就可以啦。
参考文章：http://hi.baidu.com/usen68/blog/item/e5a2370054d87d81e950cda9.html
