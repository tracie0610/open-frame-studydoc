Fedora 9 硬盘安装 配置 3D

前天因为电脑开机时间长，死机(经常出现此问题，估计电容问题，但是没见那个电容鼓起来)，好几次遇到了，还是强制关机，但是这次结果严重了！xp进不去，用磁盘软件检测结果看不到分区，甚至不认识格式，只能狠心把c盘格式化重新分区，唉，这次系统大概快半年没有安装系统了。今天linux和xp都重新安装来，linux换成fedora9，xp还是英文sp2，看着舒服，刚装完fedora9，终于看到可以用的更新源来，记下来。
今天下午比较郁闷都是刚才按照一个网页里面配置 PATH,就是更改 ~/.bash_profile,我就试着把root和普通用户下面的文件改了改，结果重启后可以看到让输入用户名和密码都界面，输入user和passwd后，闪一下，又出现这个界面，让输入user和passwd，重启电脑还是不行，当时心里真急呀。。。
    我就试着进入字符界面 Ctrl + Alt + F1 到F6 都可以，F7是进入图形界面，进入里面之后登录，可以登录，可是输入ls命令提示不识别命令，幸好还认识 “cd”，应该cd是shell的内部命令。用cd进入/bin，居然可以执行ls命令，太兴奋来，然后执行./vim ~/.bash_profile把里面都路径那句shell给改来过来，PATH=$PATH:$HOME/bin    exit退出，再登录，echo $PATH终于看到 所有都路径来，唉，虚惊一场，输入ls，终于可以执行了，幸好用过vi编辑器，对系统目录结构有基本的认识，不然真得重新安装了。。

fedora 9 硬盘安装：
1.下载Fedora-9-i386-DVD，放在FAT32分区（不能放在NTFS分区）。  
URL：http://fedoraproject.org/zh_CN/get-fedora

2.提取出isolinux文件夹下面的两个文件 initrd.img 和 vmlinuz 放在C盘根目录下。

3.下载grub4dos，解压提取出个三个文件 grldr grub.exe 和 menu.lst 放在C盘根目录下。
URL：http://download.gna.org/grub4dos/

4.用记事本打开menu.lst，最后添加上：  
title Install Fedora 9
kernel (hd0,0)/vmlinuz
initrd (hd0,0)/initrd.img

5.用记事本打开C盘根目录下的系统隐藏文件boot.ini，后面加上如下内容：  
c:\grldr=GRUB


6.重启，选择"GRUB” ，再选择"Install Fedora 9”，即可进入安装界面。
进入安装界面后，选择简体中文，英文键盘，选择硬盘安装，然后选择你下载的iso文件所在的分区，比如（hd0,4), 不幸的话都试一遍！

网络激活问题＆firefox脱机状态问题
默认情况下，我们每次进入系统都要激活一下网络才可以上网。这样太麻烦了。我们只要做一个小小的设置就可以搞定这个问题。
“系统“－”管理“－”网络“打开网络配置对话框。点击编辑按钮，把由Networkmanger控制打上勾，保存然后重启你就会发现你现在不需要再次激活网络就可以上网了。


你在首选项里把“启动Firefox时”设置为显示空白页面的话，当你下次再使用Firefox时一启动程序它就处于脱机的状态，无法上网。你只有脱机工作前面的勾去掉才可以上网。每次都这样你就会很火吧。
这是因为你一设置了这个选项为空白页，那么配置文件中的IsRelative就始终为1，所以每次打开都是脱机。知道了这个原因我们就可以着手修改了。那么这个配置文件在哪面呢？它就在/home/yeling/.mozilla/firefox这个目录下面。
yeling是我的登录用户名，当然你的可能是别的。在profiles.ini里把这个值设置为0就ok了。

fedora9 3D效果：nvidia驱动

为了3D 效果，我重新安装了一遍系统，可能是内核头文件的问题,所以最好配置好源后，不要安装其他软件，先把显卡驱动安装好。
我的显卡型号FX 5200 （AGP），刚好看到网上有人和我的显卡芯片型号一样，按照他的步骤安装：
准备工作：
1 检查内核版本 （可以省略）
uname -r
2.6.25.6-55.fc9.i686
安装中需要编译，安装 kernel-devel软件包 yum install kernel-devel   提示：Fedora 9 不包含老版本提供的kernel-source 软件包，因为要编译外部模块，只需要 kernel-devel软件包。
2 下载驱动和配置工具
NVIDIA-Linux-x86-173.14.09-pkg1.run，下载地址：
http://us.download.nvidia.com/XF ... -173.14.09-pkg1.run
你的显卡芯片型号可以在这个页面可以选择驱动 http://www.nvidia.com/Download/Find.aspx?lang=en-us

下载 nvidia-xconfig 工具 帮助设置 xorg.conf 文件。下载地址：
http://http.download.nvidia.com/ ... -xconfig-1.0.tar.gz

提示：安装驱动，需要关闭 Xserver 即退出图形模式，进入文本模式。
vi /etc/inittab 将最后的一行改为
ini:3:initdefault:          （3 是字符模式，5是图形模式）

reboot 重启电脑，进入字符模式，输入root和密码进入系统， 进入你下载nvidia-xonfig 的文件夹安装 nvidia-xconfig 设置工具，在驱动安装过程中需要用到。
tar nvidia-xconfig-1.0.tar.gz
cd nvidia-xconfig-1.0
make
make install

安装驱动 sh NVIDIA-Linux-x86-173.14.09-pkg1.run
通过安装向导可以顺利完成完成。安装后可能有提示重启，这是再编辑/etc/inittab文件，将最后一行3改为5,重启即可！
问题：
安装后进入图形界面，这时原来无法打开的桌面效果可以打开了。
选择菜单[系统]->[首选项]->[观感]->[桌面效果]，打开桌面效果窗口，选中[移动时窗口摇晃]和[立方体上的工作区]，点击[启用桌面效果]。Ctrl + Alt + 左键，拖动即可看到三维立体桌面切换效果。
某些软件中文菜单显示像乱码的问题也解决了。

提示，设置过程中很可能会compiz崩溃，所有的窗口都没有标题栏！我就遇到过两次，第一次我给重装了，这一次先不要随便动，别让电脑死机，然后打开compiz fusion icon 然后右键 Reload windows manager ,重启窗口管理器，差不多就恢复正常了。。。
compiz设置（参考）
1.启动Compiz Fusion Icon－－Setting Manager就进入3D特效的设置界面。里面的各种特效设置非常繁琐，而且每个人需要的特效又无法详尽的表述过程。

2.点选General Options--Desktop Size,在这里设置水平虚拟和垂直虚拟桌面数，3项数值可以设置为4.4.4，其他几项默认就可以了。

3.把Viewport Switcher打上勾，各项数值不用去动，默认就行了，这样就可以使用鼠标中键任意转动3D桌面并且使用滚轮方便的在窗口间任意转换。

4.点选“旋转立方体”－－启用，默认的设置已经很好用了，你只需要根据自己的喜好调整“缩放”的数值来调整立方体的景深。

5.点选“桌面立方体”－－启用，Behaviour是设置为桌面内立方体的，需要的就在“内立方体”项打勾，不需要的略过；进入Transparent Cube,其中Opacity During Rotation 是设置你启动桌面立方体的透明度，第二项是设置从立方体回到实际桌面的明暗透明度的，第三项是设置从立方体回到桌面的时间，第四项给它打勾。其他使用默认值。

6.点选特效－》动画给各种窗口的3D特效进行具体设置，在动画里面随便找到框里面的选型，点击下面的编辑，然后效果选择随机! 可以把所有的效果都选择随机，这样就可以在随机池里面自由选择一种效果了。

7.点选Cube greas,是指是否在3d里面出现齿轮！

8.进入Utility类，点选Cube Caps，这里是设置立方体顶面和底面图像明暗透明度的，你可以选择自己喜欢的图片放到立方体的顶面和底面。

9.点选“ 崩溃处理”－－启用－－启用。compiz fusion特效在实际使用中还并不能保证100％的稳定，偶尔也会自己挂掉，遇到这种情况时，你可以右键点Compiz Fusion Icon 然后选择Reload Window Manager,这样Compiz Fusion又可以重新挂载启动。同样，实际操作中还发现Emerald主题也会偶尔挂掉，方法同样是右键点Compiz Fusion Icon 然后选择Select Window Decorator,点选emerald。

10. 需要注意的是：并不是所有的3D效果都是你喜欢或者适合你的，而且开启过多的特效必然要消耗掉相当的系统资源，甚至可能引起桌面崩溃；选择适合自己电脑硬件承受能力并且不纯粹为了3D而3D，让3D特效不仅仅能让你赏心悦目而且实实在在成为你操作系统实际应用的强大助手。compiz fusion带给我的轻松和快乐便捷是其他任何系统软件都无法比拟的，甚至到了别人的电脑上也会下意识的去快速切换，没有COMPIZ绝对是一件多么不幸而痛苦的事，^_^。

11.Compiz Fusion可以自由设置的项目内容数值很多，但我用自己的体会告诉你：看不懂的地方别去乱搞，不知道它有什么作用的时候暂时别用，特别是对它默认的各种特效按键组合不要轻易的修改，因为里面有太多的按键组合了，默认的快捷键组合已经解决了与系统快捷键的关系，如果随便修改快捷键的组合有可能发生其他不可预知的错误。
