Ubuntu安装使用的初级经验

（为了便于被搜索引擎搜索到，题目都加了“ubuntu”）

一、ubuntu 安装（不用虚拟机）

方法一：光盘安装ubuntu

1. Partition Magic划分出未分配空间（最好用光盘里的工具）

2. 光盘启动安装

3. 手动分区（swap为内存大小，剩余空间挂载“/”和”/home”等，设为ext3）

保存退出

也可以从硬盘安装

方法二：Wubi安装ubuntu

虚拟光驱工具加载iso安装镜像，启动umenu.exe，选择在windows中安装。这样会安装到windows下的一个盘里

方法三：从硬盘安装ubuntu

1. Partition Magic划分出未分配空间

2. 下载ubuntu-8.04-dvd-i386.iso（用8.10版的会无法分区，虽然网上有很多解决办法，但我没有解决成功过，非常打击新手的兴趣和耐心。我的机子是amd64位的，但用校园网的客户端貌似在这个系中不管用；dvd里面有很多东西是cd里没有的，不用再下载了，比较方便）

3. 下载Grub4Dos——这是引导文件，将grldr和grub.exe放入C盘根目录；

4. 在C盘根目录新建一个 menu.lst文件，写入如下内容：

如果把 ubuntu-8.04-dvd-i386.iso放在C盘目录下，代码为

title Install Ubuntu 8.04

find_set-root  /ubuntu-8.04-dvd-i386.iso

kernel  /vmlinuz  boot=casper  find_iso=/ubuntu-8.04-dvd-i386.iso noapic

initrd  /initrd.gz

注意那个noapic，因为AMD CPU的原因，加入noapic，否则会有kernel panic；其他类型的CPU可以不加noapic

如果安装ISO放在D盘目录下“/ubuntu-8.04-dvd-i386.iso”可以改为：D:\Ubuntu 8.04-hardy-desktop-i386.iso

5. 使用虚拟光驱类软件，复制 Casper和 .disk两个目录到C盘根目录；

6. 再从Casper目录中的vmlinuz和initrd.gz文件复制到C盘根目录；

7. 修改C盘根目录下的boot.ini文件，在文件末尾加入下面代码：

C:\grldr=“GRUB”



二、ubuntu 在grub中将XP设为默认启动，方法如下：

sudo gedit /boot/grub/menu.lst

把default 0的数字改为N-1（N表示Windows被排的位置，一般为4，试试就知道了）

timeout 10 表示等待秒数，改成4



三、ubuntu 分辨率设置

方法一：

可以先联网，在“系统管理”中选择“硬件驱动”，自动下载昂达显卡驱动，安装重启后在“首选项”－“屏幕分辨率”设置里可以选择，而且此时也有了3D效果（在“外观”中设置）

方法二：

在终端里输入以下命令对文件进行编辑：

sudo gedit /etc/X11/xorg.conf

找到“Section "Monitor" ”代码段，修改成下面的信息：

Section "Monitor"

Identifier "Configured Monitor"

Option "DPMS"

HorizSync 40-83（修改显示器水平刷新频率）

VertRefresh 50-76(修改显示器垂直刷新频率，可以调整成显卡支持的最大值，比如我的设置为50-76)

End Section

（备注：保存后退出重启，你可以发现你的分辨率已经修改成1024*768，如果你还想要 1028*1024或者更高的分辨率，请看第四点。）

找到“Section "Screen"”代码段，在其中添加下列代码：

SubSection "Display"

Depth 16

Modes "1280x1024" "1024x768" "832x624" "800x600" "720x400" "640x480"

EndSubSection

SubSection "Display"

Depth 24

Modes "1280x1024" "1024x768" "832x624" "800x600" "720x400" "640x480"

EndSubSection

保存后重启，在分辨率设置的选项中便有1028*1024的分辨率。



四、Windows XP引导Ubuntu

我不喜欢从grub进windows。有一次误将ubuntu的安装空间格式化了导致连Windows也进不去了，很不爽。

1. 先安装好XP，再安装Ubuntu，此时将GRUB启动管理器安装到了MBR。

2. 启动进如Ubuntu系统，在/boot/gurb目录下找到menu.lst文件，可以将该文件复制到U盘或是直接保存到电子邮箱中。

3. 重启电脑进入XP系统，将这个文件放到C盘根目录下；同时还需要从grub4dos中复制出grldr和grub.exe两个文件，也放到C盘根目录下；修改boot.ini（如果属性为只读，先修改属性），在新行添加 c:\grldr="Ubuntu"，保存即可。

4. 重启电脑，可以启动DOS，进入命令行界面输入fdisk /mbr也可修复MBR。

5. 再次重新电脑，启动后即可实现由XP引导Ubuntu了。

6. 以后重装XP系统前只需备份boot.ini，menu.lst，grldr，grub.exe这4个文件即可再次引导Ubuntu。



五、ubuntu 中文显示和输入法

如果安装的是dvd的，在安装时选择中文，界面和输入法都支持中文。

如果安装的是cd，则用下列方法：

首选项-语言-“汉语”前打对勾，系统自动下载相关安装包，安装重启后，就成了中文的操作系统且有各种汉语输入法。

六、ubuntu音乐播放器乱码

在ubuntu下，系统是使用的UTF-8字符集。而rhythmbox等播放器有的不可以调整MP3歌曲中所包含的ID3标签信息的字符集。所以造成了乱码显示。像BMP等播放软件可以更改ID3的字符集显示，所以很方便解决乱码的问题。
用一个软件包批量修改MP3曲目中的ID3标签的字符集，使它符合UBUNTU系统下的播放器的要求。    sudo apt-get install python-mutagen
然后转到你的MP3目录，执行以全命令进行转换：

mid3iconv -e GBK *.mp3
如果需要包含子目录，可以将后缀改成如下格式：打命令的时候文件名字给 "**.mp3
最后，重新导入一次rhythmbox就OK了。

七、ubuntu视频播放器（smplayer）

1、先安装mplayer.在新立得软件管理包中选中mplayer和与它相关的包，应用安装。
2、安装解码器。
到官方网站下载w32codecs http://www.debian-multimedia.org/pool/main/w/w32codecs/

3.安装smplayer

deb包下载地址

http://www.getdeb.net/search.php?keywords=smplayer

Mplayer和realplayer都会显示中文乱码smplayer没有此问题，更像暴风影音。



八、ubuntu firefox的flash中的乱码

1. 备份49-sansserif.conf这个字体配置文件

sudo cp /etc/fonts/conf.d/49-sansserif.conf /etc/fonts/conf.d/49-sansserif.conf.bak

2. 删除49-sansserif.conf文件做完后重启firefox就可以了。

sudo rm /etc/fonts/conf.d/49-sansserif.conf
