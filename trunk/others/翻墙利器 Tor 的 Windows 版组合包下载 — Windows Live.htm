    翻墙利器 Tor 的 Windows 版组合包下载

          大名鼎鼎的翻墙利器 TOR 现在有 Windows 图形版了（TOR+Privoxy)，它支持所有的Windows系统，98、98SE、NT4、2000、XP、Vista、Server。支持X86,X64位系统。你可以到它的主页 http://tor.eff.org/ 下载，(但是现在TOR的主页已经被和谐了）我在这里提供TOR的镜像网站 https://tor.plentyfact.net/mirrors.html.zh-cn，打开时你可以看到“此网站的安全证书有问题。”等字样，不用管它，点击“继续浏览此网站(不推荐)。”就可以打开网页了。打开后请选择“https dist”下载，因为https 站点是加密浏览封锁不了的。

     

    这里提供两个https站点，你可以直接打开
    https://torproj.xpdm.us/
    https://tor.plentyfact.net/index.html.zh-cn


    下载后点击程序安装

    点击Next

    点击Next

    点击Next

    点击Next

     

    点击Next

    点击finish安装结束

    点击Setting打开程序的设置界面，再点击Appearance，在下拉菜单中选择“简体字”，按确定。

    退出程序后,再次启动Vidalia程序，就会显示中文界面了

    虽然是中文界面，但是还是有一些英文没有翻译，点击“设定”可以看到两行英文，

    上面一行的意思是：系统启动时运行Vidalia，为了节省系统资源，可以把这一行前面的勾去掉，我们不需要电脑启动时就运行Vidalia。

    下面一行的意思是：当启动Vidalia时运行Tor,这一行前面的勾请保留。

    以后只要你启动Vidalia，你就可以看到在你电脑的右下角有“洋葱头图标”，当那个“洋葱头”图标从黄色变成绿色的时候，TOR就已经联入了网络，

    这时，你只要右键点击桌面上的浏览器图标，选择“属性”

    点击“连接”选项牌------再点击“局域网设置”

    在“代理服务器”里面设置你的浏览器代理为 127.0.0.1，端口 8118，如下图所示：

    点击“确定”就可以了。这时候你就能“翻墙”了。你可以打开你的浏览器浏览世界上任何的网站，而且你的浏览全程是加密的。

    当你打开网站时产生流量时，你可以看到屏幕右下角的“P”字会像雷达一下的转动。如下图所示：

    浏览结束点击“停止”你可以停止 TOR 的运行。停止后别忘了修改回你的浏览器的代理设置啊。

          虽说是 Windows 版，其实还是 DOS 下的程序，只是加了一个 Windows 外壳罢了，但是易用性大大提高了。

    以下还有两个细节请注意：

    （一）浏览敏感网站时请先清除你的浏览器cookies

    谨防 cookies 泄密：如果你曾在未使用 Tor 和 Privoxy 的情况下浏览了某个站点，这个站点给你设置了一个 cookie，那么，即使你又开始使用 Tor，那个 cookie 仍能用来确认你的身份。所以你浏览敏感网站前，请清理浏览器的 cookies。

    （二）你的浏览器插件也可能会泄漏你的上网信息

    浏览器插件例如 Java、Flash、ActiveX、RealPlayer、Quicktime、Adobe 的 PDF 插件等等能被用来泄漏您真实的 IP 地址。当然你可以使用两个浏览器，（一个使用 Tor 浏览敏感网站，另一个用作平时一般的浏览）。

    现在的64位操作系统提供了两个IE浏览器，一个是32位的 IE,一个是64位的IE。一般我们都用32位的浏览器，因为现在很多的网站上的插件不支持64位IE，所以我们可以用32位的IE浏览器浏览普通网站，用64位的IE浏览敏感网站。不是一举两得吗。

    （三）千万不要安装国内网站上的浏览器插件

    当你刚装完一个新的系统浏览网站时，；浏览器常常会弹出一个对话框要求你安装插件，请务必上国外的官方网站安装该浏览器插件。例如：上Adobe网站安装其Flash插件，等等。

    在浏览国内某些网站时，也常会弹出对话框要求你安装某某插件等等，请千万不要安装，这些插件常常是被改装过的，轻者会使你的浏览器无辜崩溃，重者你的信息被窃取你都蒙在鼓里呢。切记，切记！

    推荐 TOR的改进版------torplus

    有时候Tor 的 Windows 版组合包打开网站很慢，或者打开网站失败。请试试这个小玩意儿torplus，条件是你已经安装了Tor 的 Windows 版组合包。因为torplus.exe它也要用到Tor 的 Windows 版组合包里的一个软件Privoxy，就是屏幕右下角的那个“P”。

    点击下载  torplus

    这个软件不需要安装。点击 torplus.exe 就可以了

           

    当连接成功后，设置你的浏览器代理为 127.0.0.1，端口8118就可以翻墙。至于为什么Tor 的 Windows 版组合包上不去的时候，torplus还能翻墙，留给你自己琢磨吧。呵呵。
